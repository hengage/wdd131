Henry Chizoba's Portfolio

A personal website showcasing my work, skills, and career as a software engineer.

Site Purpose

This site will present my background, skills, and projects in a structured, accessible way. It will serve both as a course project and a professional tool for recruiters, collaborators, and clients. It also demonstrates responsive design and modern web development practices.

Scenarios

  • What projects has Henry worked on, and what technologies did he use?
  • How can I contact Henry or view his GitHub/LinkedIn profile?

Color Schema

  • Primary Color: Classic dark navy (#001F3F) - headings and navigation.
  • Accent Color: Bright Cyan (#06B6D4) - highlights, buttons, and links.
  • Background: Light Gray (#F3F4F6) - clean, neutral backdrop.

Typography

  • Headings: Montserrat (bold, modern, professional).
  • Body Text: Roboto (clean, highly readable).
  • Accent Font: Fira Code (for code snippets/projects).
